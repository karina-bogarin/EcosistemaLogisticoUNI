# Sistema de Ecosistema Logístico Inteligente (POO)

Este proyecto consiste en el diseño e implementación de un sistema de gestión de flota de vehículos para una empresa de logística. Se aplica la **Programación Orientada a Objetos (POO)** para modelar diferentes tipos de transporte como camiones, drones y motos de delivery, gestionados desde un centro de control centralizado.

# Estructura del Proyecto
El sistema se basa en los siguientes pilares de la POO:
1. **Abstracción:** Uso de la clase abstracta `Vehiculo` y la interfaz `IConectable`.
2. **Herencia:** Clases `CamionAutonomo`, `DronTransporte` y `MotoDelivery`.
3. **Polimorfismo:** Procesamiento dinámico de la flota mediante el método `monitorearFlota()`.
4. **Encapsulamiento:** Atributos privados con métodos de acceso controlados.
